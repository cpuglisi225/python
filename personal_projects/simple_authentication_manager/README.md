# Authentication Manager

A simple authentication system prototype written in Python.

The project demonstrates basic concepts of:

- password hashing using SHA-256
- file handling
- user registration and login
- basic authentication flow

## Features

- Register new users
- Store hashed passwords locally
- Authenticate existing users
- Simple terminal interface

## Project Structure

```
Authentication-Manager/
│
├── main.py
├── password_db.txt
└── README.md
```

## Usage

Run the program:

```
python main.py
```

Choose an option from the menu:

```
[1] Register
[2] Login
[q] Quit
```

## Notes

This project is for educational purposes and demonstrates the fundamentals of authentication and password hashing.

It is not intended to be used as a production authentication system.
